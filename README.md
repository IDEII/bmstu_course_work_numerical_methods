# Курсовая работа по курсу "Численные методы" в МГТУ им. Н. Э. Баумана
## Моделирование распространения болезни
* Агенты (люди) имеют различные состояния здоровья (восприимчивый, латентный, субклинический, скрытозараженный, явно зараженный дома/в больнице, иммунный, умерший).

* Агенты разделены на социальные группы (школьники, студенты, работники разных сфер, пенсионеры) с разным расписанием и количеством контактов.

* Модель учитывает случайные контакты, внутрисемейное и рабочее взаимодействие, а также факторы внешней среды, такие как введение/снятие локдауна и уровень доверия к вакцинации.

## Калибровка параметров:


* Для точной настройки модели под реальные эпидемиологические данные (по городу Москва) используется метод роя частиц (PSO).

* Этот метод оптимизирует ключевые параметры болезни (заразность, летальность, продолжительность иммунитета и др.) для минимизации расхождений между моделированием и статистикой.

## Реализация и визуализация:

* Вычислительное ядро модели написано на Golang для обеспечения высокой производительности при работе с популяцией в 10 млн. агентов.

* Для визуализации результатов используется Python с библиотеками matplotlib и pandas, что позволяет строить графики динамики заболевания и сравнивать их с реальными данными.

## Результаты:

* Модель показала хорошую сходимость с реальной статистикой по заболеваемости и смертности в Москве, а также адекватно оценила долю скрытых случаев.
